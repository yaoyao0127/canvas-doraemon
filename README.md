canvas练习
=====================================
用H5引入的canvas画布绘制一个哆啦A梦=￣ω￣= 
定义了一个小函数来实现在网页上实时显示光标的坐标辅助画图，思路如下  
1.为body标签绑定一个鼠标移动事件，并与坐标函数挂钩  
2.在脚本区域自定义坐标函数，并分别定义xy并赋值  
3.捕获要显示坐标的div，用innerHtml方法来输出显示内容
